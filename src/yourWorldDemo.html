User Registration and Authentication

Users can register with password encryption and email verification.

Upload a Profile Picture

Users can can upload, crop and save a profile picture. Profile picture can be updated at any time.

Log out and Log Back In, All Data Retained

User can log out without losing profile picture or any other data. Username and password is required to log back in.

Create a Post with User Specific Delete Button

Users can create a post of whatever length. The post appears in real time for all users. Only the user that created the post has the ability to delete their post.

'Like' a Post

Users can 'Like' a post. Displays in real time for all users.

Comment on a Post with User Specific Delete Button

Users can comment on a post. The comment appears in real time for all users. Only the user that created the comment has the ability to delete their comment.

'Like' a Comment

Users can 'Like' a comment. Displays in real time for all users.

Edit Profile

Users can edit their profile. Edits reflect in real time for all users.

Delete Profile

Users can delete their profile, which also deletes all of their posts and comments. Required to re-register to be able to access the app again.
