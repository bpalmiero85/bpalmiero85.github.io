Start a Game

Users can start a new game by clicking the "Play" button. The game starts with a 30-second timer during which the user can smash moles and avoid monsters to score points.

Smash Moles

Users can smash moles that appear in random holes. Each mole hit increases the user's score by 1 point. The goal is to smash as many moles as possible within the given time.

Avoid Monsters

Users should avoid hitting monsters that occasionally appear instead of moles. Each monster hit decreases the user's score by 2 points. Hitting a monster can potentially end the game if the score drops to zero.

View Timer

The remaining game time is displayed at the top of the screen. The game ends when the timer reaches zero, and the user's final score is displayed.

Restart Game

Users can restart the game after it ends by clicking the "Play Again" button. This resets the score and the timer for a new game session.

Top Scores

Users can view the top three high scores achieved by players. This leaderboard updates automatically when a new high score is recorded.

Player Info

Users can view their username and current high score. The username is fetched upon register/login, and the score is fetched upon game end.
