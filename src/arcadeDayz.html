User Actions

User Registration and Login

Users can register for a new account or log in to an existing one. The registration ensures unique usernames, and passwords are securely hashed.

Playing the Whac-A-Mole Game

Users can start a game of Whac-A-Mole, where they have 2 minutes to smash as many moles as possible while avoiding monsters. Moles add points, and monsters subtract points.
